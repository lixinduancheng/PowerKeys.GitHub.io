## `Win` 键增强

> **一键完成常见的 Windows 操作！**

<br>

| 随同 `Win` 键按下                                                  | 功能                                     |
| :----------------------------------------------------------------- | :--------------------------------------- |
| **`+`<br>`-`<br>`\`**|**增大音量<br>减小音量<br>静音（取消静音）** |
| **`[`<br><big>`"`</big><br>`]`**                                   | **上一首歌<br>暂停（继续）<br>下一首歌** |
| **`Shift + PrtSc`**                                                | **启动 Windows 截图工具**                |
| **`0`**                                                            | **启动计算器**                           |
| **`Delete`<br>`Shift + Delete`**                                   | **打开回收站<br>清空回收站**             |
| **`End`**                                                          | **休眠 Windows**                         |
| **`Page Up`<br>`Page Down`<br>`Esc`**                              | **放大屏幕<br>缩小屏幕<br>退出放大镜**   |
